# Fork of autoComplete.js :sparkles:

## Get Started

### Clone:

-   Clone autoComplete.js to your local machine

```shell
git clone git@github.com:pewtrusts/autoComplete.js.git
```

## Edit
```shell
npm run dev
```

## Build
```shell
npm run build
```
